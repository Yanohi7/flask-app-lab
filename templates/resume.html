Про себе

Я веб-розробник з більш ніж 5-річним досвідом у галузі створення високоякісних, динамічних веб-додатків та сайтів. Моя кар'єра розпочалася з невеликих проектів, і з часом я освоїв не лише основи веб-розробки, але й сучасні інструменти та фреймворки, що дозволяють створювати масштабовані та ефективні рішення. Я спеціалізуюся на бекенд-розробці з використанням Python і Flask, але також маю глибокі знання у фронтенді, використовуючи HTML, CSS і JavaScript для створення інтерактивних інтерфейсів користувача. Моя мета — постійне вдосконалення та впровадження нових технологій для створення інноваційних рішень.

Мені подобається вирішувати складні технічні задачі, працювати над оптимізацією коду та розробляти рішення, що забезпечують зручний користувацький досвід. Я прагну працювати над проектами, що мають реальний вплив на бізнес і користувачів, використовуючи сучасні інструменти та кращі практики програмування.

Освіта

Закінчив Прикарпатський національний університет імені Василя Стефаника, де отримав ступінь бакалавра в галузі інформаційних технологій. Під час навчання я зосередився на вивченні основних принципів програмування, алгоритмів та структур даних, а також на технологіях веб-розробки. Окрім академічних курсів, я активно брав участь у проектах, де мав можливість застосовувати свої знання на практиці. Моя остання курсова робота була присвячена розробці mySQL сервера.

Постійне самонавчання є важливою частиною моєї професійної діяльності. Я регулярно проходжу онлайн-курси та семінари, пов'язані з новими технологіями та фреймворками, такими як Docker, Kubernetes, React, та інші.

Навички

  • HTML, CSS, JavaScript: Глибокі знання фронтенд-розробки, включаючи адаптивну верстку, оптимізацію для SEO та інтерактивні елементи. Використовую сучасні бібліотеки та фреймворки, такі як Bootstrap та React, для створення сучасних інтерфейсів користувача.
  • Python, Flask, Django: Основний стек для бекенд-розробки, з акцентом на розробку RESTful API, інтеграцію з базами даних та обробку запитів. Досвід у побудові складних веб-сервісів та роботи з мікросервісною архітектурою.
  • MySQL, PostgreSQL: Робота з реляційними базами даних, розробка складних SQL-запитів, оптимізація продуктивності та забезпечення безпеки даних.
  • Git та CI/CD: Управління версіями коду за допомогою Git, налаштування систем автоматичного деплою та інтеграції (GitHub Actions, Jenkins) для забезпечення безперервної доставки програмного забезпечення.
  • Docker та контейнеризація: Використовую Docker для контейнеризації додатків, що дозволяє розгортати їх на різних платформах без зміни налаштувань середовища.
  • Основи безпеки веб-додатків: Захист додатків від основних загроз, таких як SQL-ін'єкції, XSS та CSRF атаки, впровадження багаторівневого захисту.

Досвід роботи

  • Проект 1: Розробка веб-додатку для управління завданнями. Цей додаток був побудований для внутрішнього використання у компанії та дозволяв відстежувати проекти, завдання та їх статуси. Я відповідав за розробку бекенду на Flask, створення бази даних та реалізацію логіки завдань, а також інтеграцію з фронтенд-інтерфейсом через RESTful API. Після впровадження додатку продуктивність команди зросла на 30% завдяки автоматизації процесів управління завданнями.
  • Проект 2: Інтеграція платіжних систем у веб-сайти. Працював над інтеграцією популярних платіжних платформ (Stripe, PayPal) для інтернет-магазину. Цей проект вимагав глибоких знань у роботі з API, безпеки платіжних операцій та захисту даних користувачів. Розробив повний цикл платіжної обробки, включаючи верифікацію транзакцій та захист від шахрайства. В результаті магазин отримав надійний та зручний спосіб обробки онлайн-платежів, що збільшило конверсію покупок на 15%.
  • Проект 3: Розробка соціальної платформи для взаємодії користувачів. Я був відповідальний за створення бекенду платформи, що дозволяла користувачам реєструватися, створювати профілі, ділитися контентом та взаємодіяти через чати та коментарі. Для забезпечення масштабованості проекту використовував мікросервісну архітектуру та обробку асинхронних запитів за допомогою Celery. Платформа успішно витримала навантаження у понад 10 тисяч активних користувачів у день після запуску.
